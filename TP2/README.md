# TP2 - Chatbot Básico

## Descripción
Chatbot simple que responde a preguntas básicas usando un diccionario de respuestas predefinidas.

## Uso

```bash
python chatbot_basico.py
```

## Comandos disponibles
- `hola` - Saludo
- `como estas` - Preguntar estado
- `que eres` - Información del bot
- `tu nombre` - Nombre del bot
- `ayuda` - Ver comandos
- `chiste` - Escuchar un chiste
- `adios` / `gracias` - Despedida
- `salir` - Terminar programa

## Características
- ✅ Respuestas predefinidas
- ✅ Reconocimiento de palabras clave
- ✅ Interfaz por consola
- ✅ Manejo de entrada del usuario
