# TP3 - Servidor Node.js

## Instalación
```bash
npm install
```

## Uso
```bash
npm start
```

Servidor: `http://localhost:3000`

## Rutas API
- `GET /` - Página principal
- `GET /api/info` - Info del servidor
- `GET /api/saludo/:nombre` - Saludo personalizado
- `GET /api/hora` - Hora actual
- `GET /api/datos` - Datos de ejemplo
