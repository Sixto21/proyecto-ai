# TP5 - Chatbot Mejorado con IA

## Descripción
Chatbot avanzado con procesamiento de lenguaje natural, análisis de sentimientos y contexto.

## Uso
```bash
python chatbot_mejorado.py
```

## Características
- ✅ Procesamiento de lenguaje natural (NLP)
- ✅ Extracción de información (nombres)
- ✅ Análisis de sentimientos
- ✅ Base de conocimiento
- ✅ Contexto personalizado
- ✅ Expresiones regulares
